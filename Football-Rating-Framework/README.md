# Football Performance Model

This project implements a Football Performance Model using Streamlit, allowing users to manage player and match data, calculate performance metrics, and visualize results through an interactive web interface.

## Features

- **Player Management**: Add, edit, and view player information.
- **Match Management**: Record match details and results.
- **Performance Metrics**: Calculate and display various performance metrics, including Match Performance Ratings (MPR) and Contextual Season Ratings (CSR).
- **Action Logging**: Log decision points during matches to analyze player performance.

## Requirements

To run this application, you need to have Python 3.x installed along with the following packages:

- Streamlit
- pandas
- numpy

You can install the required packages using the following command:

```
pip install -r requirements.txt
```

## Usage

1. Clone the repository:
   ```
   git clone https://github.com/matheusc5kuhn-afk/Football-Rating-Framework.git
   ```

2. Navigate to the project directory:
   ```
   cd Football-Rating-Framework
   ```

3. Install the dependencies:
   ```
   pip install -r requirements.txt
   ```

4. Run the application:
   ```
   streamlit run app.py
   ```

5. Open your web browser and go to the URL provided in the terminal to access the application.
